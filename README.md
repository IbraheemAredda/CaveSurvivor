<h2>Cave Survivor</h2>
<p>A prototype of a turn-based top-down shooter, the main purpose of this game was to implement Path-finding algorithms like <b>Djikistra</b> and <b>A*</b>, and getting familiar with them.</p>

<h2>Game Mechanisms</h2>
<p>It contains some basic common mechanisms such as:</p>
<ul>
  <li>Moving around</li>
  <li>Shooting</li>
  <li>Health/Hunger/Ammo System</li>
</ul>

<h2>Development Tools</h2>
<p>You will need <b>Unity3D</b>, any version above 5.6.6f2, in order to be able to open up this project</p>
